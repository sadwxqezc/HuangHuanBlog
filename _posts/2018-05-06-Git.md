### Git Tag

Tag是打标签，可用来作为发布版本管理

```
#显示已有标签
git tag

#打标签
git tag -a tagName -m comment
git tag -a 0.5.5 -m '发布版 0.5.5'

#显示tag信息
git show 0.5.5

#推到服务器
git push origin 0.5.5

#本地删除和远程删除
git tag -d 0.5.5
git push origin :refs/tags/0.5.5
```



### Git Reset

重置

```
#本地回滚到指定commit
git reset <commit>

#本地reset后远程同步
git push -f origin master
```
